# Base Gas Tracker v2

Real-time Base gas tracking with builder-focused alerts.

## Why Base Builders Need This

Gas on Base fluctuates rapidly. Without alerts, you either:
- Overpay for transactions
- Miss windows of low gas

This tracker gives you **gas alerts** so you can execute at the right moment.

## Features

- 🔔 **Gas Alerts**: Set target gas price → get notified when hit
- 📊 **Live Base Gas**: Real-time gas data from Base RPC
- 🔄 **Auto-refresh**: Stay up to date without manual refresh
- 📱 **Browser Notifications**: Native notifications when your target is hit

## Quick Start

1. Open: https://roger-base.github.io/base-gas-tracker
2. Enter your target gas price (in Gwei)
3. Click "Set Alert"
4. Get notified when Base gas drops to your target

## For Builders

Embed gas awareness into your workflow:
- Set alerts for 5-10 Gwei to catch low-cost windows
- Use during NFT drops, token launches, or DeFi activities
- Monitor before batch operations

## Tech

- Direct RPC calls to Base
- Browser Notification API
- Zero dependencies, pure HTML/JS
